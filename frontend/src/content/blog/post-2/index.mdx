## Instlación de WSL2
1.**Habilitación de WSL**

Si eres desarrollador y usas el sistema operativo Windows, te recomendamos instalar Windows Subsystem for Linux, mejor conocido como WSL o WSL2. 
Ejecute el siguiente comando:

```bash
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
```

2.**Habilitación de Virtualización**

Antes de instalar WSL2, debemos habilitar las características de virtualización de nuestra computadora, para esto debes abrir nuevamente tu terminal y debes ejecutar el siguiente comando:

```bash
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
```


3.**Instalación de WSL2*

Abra Powershell como administrador, y ejecutar el siguiente comando. 

```bash
wsl --install
```

4.**Instalación de tu distribución de Linux favorita*

Abre Microsoft Store, el cual viene instalado por defecto en Windows. Busca e instala la distribución de Linux favorita, te recomiendo que instales Ubuntu 22.04.2 LTS (puedes escoger otra distribución Linux como Fedora o Debian).

En Microsoft Store seleccionamos la distribución de nuestra preferencia y seleccionamos Obtener.

5.**Reinicia tu Computador**
Finalizar el proceso reiniciando tu computadora para asegurar que la distribución de Linux se instale correctamente. Luego, puedes acceder a tu nueva terminal Linux y configurar tu usuario y contraseña.

## Conclusión

Con estos pasos, ahora estás listo para aprovechar todo lo que Linux tiene para ofrecer en un entorno Windows, lo que te permitirá realizar tareas de desarrollo y administración de sistemas de manera más versátil y eficiente. ¡Disfruta de tu experiencia con WSL!
